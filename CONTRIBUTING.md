# Contributing

## Running locally

```bash
git clone https://github.com/satyaborg/specloop.git
cd specloop
chmod +x specloop
./specloop "your feature description"
```

## Prerequisites

- [gum](https://github.com/charmbracelet/gum) — interactive prompts
- [glow](https://github.com/charmbracelet/glow) — terminal markdown rendering
- [Claude Code](https://docs.anthropic.com/en/docs/claude-code) — AI coding agent

## Submitting changes

1. Fork the repo and create a feature branch
2. Make your changes
3. Run `shellcheck specloop` and fix any warnings
4. Open a PR with a clear description of what changed and why
